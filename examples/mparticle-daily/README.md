# mparticle Daily Intelligence (Sanitized Public Example)

This folder contains a GitHub-safe public packaging of a scheduled content intelligence workflow.

## What this example demonstrates
- scheduled content sync and scan workflow
- article content extraction and normalization
- digest generation logic
- scheduled delivery setup examples
- configuration templating for public sharing

## Included materials
- `scripts/`: sanitized workflow scripts and entry points
- `config/`: public-safe config templates
- `data/sample_articles.json`: synthetic sample output structure
- `deploy/launchd/`: example macOS launchd jobs for scheduled execution

## Intentionally omitted
- real credentials / cookies / auth state
- real webhook targets
- real hostnames / private infrastructure
- local databases
- private monitored accounts and raw production content
